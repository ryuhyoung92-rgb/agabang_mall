### 어려운점 & 자주하는 실수
* 실제 홈페이지를 제대로 확인하지 않고 작업하는 습관이 있음 -> 지양해야할 점
* 그룹으로 묶이는 부분을 더 명확히 파악하기
* id, class에 대문자 사용하지 않기(이름 확인 잘하기)
* 자주 사용되는 태그들에는 class 붙여주기
* `<a><h1>` (X) -> `<h1><a>` (O) h태그를 a태그보다 우선순위로 적기
